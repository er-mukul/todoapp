To do app

Technologies used:
   1) Backend:
      Spring boot,In memory db H2,Maven
   2) Frontend
      Jsp,jQuery,css,html,javascript,jstl

Pre-requisite:
Java 1.8 and Maven

How to Run:
Run commands in the project directory:
1) mvn clean install
2) mvn spring-boot:run


Application will be available at http://localhost:8080/login
